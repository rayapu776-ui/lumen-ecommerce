# lumen-ecommerce
LUMEN is a full-stack e-commerce web application I built as my second-year project. It includes a complete product catalog with search and filtering, persistent shopping cart, real Stripe payment integration, secure authentication with JWT and bcrypt, customer order history, and a role-protected admin dashboard for managing products and orders.
Built with Next.js 14 (App Router), Tailwind CSS, and SQLite — designed to run with zero external services so it's easy to clone and try.
