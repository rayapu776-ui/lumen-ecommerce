# LUMEN – Full-Stack E-Commerce Platform

## Overview
LUMEN is a modern full-stack e-commerce web application built with Next.js 14. It provides a complete online shopping experience with authentication, product management, shopping cart, checkout, and an admin dashboard.

## Features
- User Authentication (JWT)
- Product Catalog
- Shopping Cart
- Stripe Payment Integration
- Order History
- Admin Dashboard
- Responsive Design

## Technologies Used
- Next.js 14
- React
- Tailwind CSS
- SQLite
- better-sqlite3
- JWT
- bcryptjs
- Stripe

## Installation

bash
npm install
npm run dev
